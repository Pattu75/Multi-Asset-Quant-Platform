# Multi-Asset Quantitative Trading & Portfolio Analytics Platform

Built by Zakariya Boutayeb — Quantitative Finance & Data Science

## Overview
This project is an interactive multi-asset market research and portfolio analytics platform built with Python and Streamlit. 
It enables users to search and analyze financial data across equities, ETFs, bonds, commodities, REITs, indexes, and selected crypto proxies; 
compare assets against benchmarks and optimize portfolios for historical Sharpe ratio; 
generate BUY / HOLD / SELL signals and backtest systematic allocation strategies.

## Core Features
- Market Research Terminal with ticker search
- Multi-asset universe across major asset classes
- Market data explorer with benchmark comparison
- Multi-asset comparison and correlation analysis
- Portfolio optimizer for highest historical Sharpe ratio
- BUY / HOLD / SELL signal dashboard
- Strategy backtester with volatility targeting and regime filters
- Portfolio growth, drawdown, and rolling Sharpe analytics
- Downloadable CSV outputs
